# User Guide for Construction Project Management DB

## Introduction
This document serves as a complete user guide for managing construction projects using the Construction Project Management Database (CPM DB). It includes instructions on adding organizations, creating projects, managing apartment units, assigning resources, entering time, submitting invoices, tracking performance metrics, and generating reports.

## Adding Organizations
To add organizations, follow these steps:
1. Navigate to the Organizations section in the menu.
2. Click on 'Add Organization'.
3. Enter the required information, including name, address, and contact details.
4. Save the changes to add the organization to the database.

## Creating Projects
Steps to create a new project:
1. Go to the Projects section.
2. Click on 'Create New Project'.
3. Fill in the project details such as title, description, and deadline.
4. Assign the organization associated with the project.
5. Click 'Create' to save the project.

## Managing Apartment Units
To manage apartment units within a project:
1. Select the project from the Projects list.
2. Navigate to the Apartment Units tab.
3. Click 'Add Unit' to create a new apartment unit, or select an existing one to edit.
4. Enter details like unit number, size, and status, then save your changes.

## Assigning Resources
Follow these steps to assign resources to a project:
1. Open the project in need of resource allocation.
2. Go to the Resources section.
3. Click on 'Assign Resources'.
4. Select from the available resources and specify their roles.
5. Confirm the assignments by clicking 'Save'.

## Entering Time
To enter time for tasks:
1. Navigate to the Time Tracking section.
2. Select the project and task you want to log time for.
3. Enter the hours worked and any relevant notes, then click 'Submit'.

## Submitting Invoices
Steps to submit invoices:
1. Access the Invoices section.
2. Click on 'Create Invoice'.
3. Fill out the invoice details including amounts and project reference.
4. Submit the invoice for approval.

## Tracking Performance Metrics
To track project performance metrics:
1. Go to the Performance Metrics tab within a project.
2. Review the listed key performance indicators (KPIs).
3. Adjust project management strategies based on these metrics as necessary.

## Generating Reports
To generate reports:
1. Navigate to the Reporting section.
2. Choose the type of report you need (e.g., project summary, financial report).
3. Customize any filters and select the 'Generate Report' button.
4. Download or view the report as required.

## Conclusion
This guide provides the necessary steps to efficiently manage projects within the CPM DB framework. For further assistance, please refer to the help section or contact support.
