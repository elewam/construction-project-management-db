# SETUP GUIDE

## Prerequisites
- Microsoft Access must be installed on your computer.
- Familiarity with basic database concepts is helpful.

## Step 1: Creating Tables
1. Open Microsoft Access.
2. Go to the 'Create' tab.
3. Click on 'Table Design'.
4. Define the fields (columns) for your table as required for your project.
5. Save each table with a meaningful name.

## Step 2: Importing Sample Data
1. In Access, go to the 'External Data' tab.
2. Choose the source of your sample data (e.g., Excel, CSV).
3. Follow the import wizard steps to import data into your created tables.

## Step 3: Setting Up Relationships
1. Go to the 'Database Tools' tab.
2. Click on 'Relationships'.
3. Add the tables you want to create relationships between.
4. Drag and drop fields to create relationships, specifying referential integrity as needed.

## Step 4: Creating Queries
1. Navigate to the 'Create' tab.
2. Click on 'Query Design'.
3. Select the tables you want to query.
4. Choose the fields you want to display and define any criteria for your query.
5. Save your query with a descriptive name.

## Step 5: Building Forms
1. Go to the 'Create' tab.
2. Click on 'Form Design'.
3. Choose the table or query that your form will be based on.
4. Use the design tools to add fields, labels and controls to your form.
5. Save the form with a meaningful name.

## Additional Notes
- Ensure you back up your database regularly.
- Refer to Access documentation for advanced features.
